Redefinir Senha (Telefone/Email)

Use esta função para redefinir a senha de um usuário via Telefone (SMS) ou Email (Supabase Auth).

Lógica de Redefinição de Senha

Como funciona:

  1. Usuário solicita redefinição no app mobile
  2. Sistema gera OTP e envia por SMS
  3. Usuário insere o código no app
  4. App valida o código via /api/otp/verify
  5. App envia nova senha via /api/password/reset
  6. Senha é atualizada no banco de dados

Lógica de Criação de Usuário e Envio de OTP

Fluxo de Registro:

  1. Usuário insere telefone no app mobile
  2. App chama /api/register com telefone e senha
  3. Sistema verifica se telefone já existe
  4. Senha é hasheada com bcryptjs
  5. Usuário é criado na tabela usuarios
  6. Sistema gera OTP automático
  7. OTP é enviado por SMS via Twilio
  8. Usuário recebe código e verifica via /api/otp/verify
